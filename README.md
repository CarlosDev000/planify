# Planify - Todo App

A simple and efficient todo application to help you organize your tasks and boost productivity.

## Features

- Create, edit, and delete tasks
- Organize tasks with categories and priorities
- Set due dates and reminders
- Sync across devices
- Dark mode support

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
git clone https://github.com/yourusername/planify.git
cd planify
npm install
```

### Running the App

```bash
npm run dev
```
